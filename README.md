# sistema_parqueadero
El siguiente codigo simula un sistema de parqueadero que ademas de: 
Generar un mapa, mostrarlo con vias y denotando entrada y salida; muestra espacios libres y ocupados 
Realiza el registro de vehiculos por placa y registra la hora de ingreso manulamente
Y que al momento de darle salida a un vehiculo calcula el pago por hora.

Tambien:
Asigna lugares automaticamente dependiendo del vehiculo
Realiza la validacion de la placa en la que identifica si es carro o moto, 
verifica que la cantidad de caracteres sea correcta y valida que no sea una placa repetida
Muestra el historial de las ultimos 100 salidas donde guarda la placa y el pago
Al momento de mostrar el mapa muestra una lista con la ubicacion, la placa y el tiempo que lleva el vehiculo
